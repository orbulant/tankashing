# BigOrder

<img src="/bigorder.png" alt="BigOrder Login Page" width="350" />

A **Full-Stack (MERN)** web application for small-medium food establishments that allows customers to place orders online via the website after scanning a QR Code.

Tech Used: MongoDB, ExpressJS, ReactJS, NodeJS, Post Deployment Scripts, Git

[Click here to visit BigOrder hosted on Heroku!](https://bigorder.herokuapp.com/)
